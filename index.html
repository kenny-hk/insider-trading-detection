S&P 500 Insider Trading API

This API provides access to insider trading data from SEC Form 4 filings for S&P 500 companies. All data is updated daily through automated GitHub Actions workflows.

API Endpoints

Get All Companies

GET /data/json/companies.json

Returns a list of all companies with metadata including the number of transactions and date ranges.

{
  "last_updated": "2024-04-15T10:30:45.123456",
  "count": 20,
  "companies": [
    {
      "ticker": "AAPL",
      "name": "Apple Inc.",
      "transaction_count": 42,
      "latest_transaction": "2025-04-10",
      "earliest_transaction": "2024-10-15"
    },
    ...
  ]
}

Get Latest Transactions for a Company

GET /data/json/{ticker}/latest.json

Returns the 50 most recent insider trading transactions for the specified company.

{
  "ticker": "AAPL",
  "last_updated": "2024-04-15T10:30:45.123456",
  "count": 50,
  "trades": [
    {
      "id": 123,
      "issuer_name": "Apple Inc.",
      "reporting_owner": "Cook, Tim",
      "reporting_owner_position": "Chief Executive Officer",
      "transaction_date": "2025-04-10",
      "transaction_shares": "10000",
      "transaction_price": "180.25",
      "transaction_type": "S",
      "shares_after_transaction": "845000"
    },
    ...
  ]
}

Get Summary Data

GET /data/json/summary.json

Returns summary data including the largest transactions by value and the most recent transactions across all companies.

  "last_updated": "2024-04-15T10:30:45.123456",

    ...
  ]
}

Usage Examples

JavaScript Example

// Fetch all companies
fetch('https://yourusername.github.io/insider-trading-detection/data/json/companies.json')
  .then(response => response.json())
  .then(data => {
    console.log(`Found ${data.count} companies`);
    // Process the data
  });

Python Example

import requests

# Fetch latest transactions for Apple
response = requests.get('https://yourusername.github.io/insider-trading-detection/data/json/AAPL/latest.json')
data = response.json()

print(f"Found {data['count']} transactions for {data['ticker']}")
for trade in data['trades']:
    print(f"{trade['transaction_date']}: {trade['reporting_owner']} {trade['transaction_type']} {trade['transaction_shares']} shares at ${trade['transaction_price']}")

Data Update Schedule

The insider trading data is updated daily at 8:00 UTC through an automated GitHub Actions workflow. Each update fetches the latest Form 4 filings from the SEC EDGAR database for 20 S&P 500 companies.
